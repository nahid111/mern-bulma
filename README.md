# mern-shop
<p>
MERN stack Boilerplate
</p>

## installation
- install NodeJs & npm
- install MongoDB
- install dependencies 
```
$ npm install
```
- Copy & rename .env.example file to .env
- Set env variables

```env
NODE_ENV=development
PORT=5000
MONGO_URI=
JWT_SECRET=
```
- Seed the DB using
```
$ node seeder -i
```
- Run in Dev mode
```
$ npm run dev
```
